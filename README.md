# Python Basics Scripts 🐍

Коллекция моих первых программ и скриптов, написанных в процессе изучения основ языка Python. Эти задачи помогли мне освоить синтаксис, логику алгоритмов и чистый код перед переходом к веб-фреймворкам и ботам.

## 📂 Что внутри?

В этой папке собраны решения классических задач и тренировочные скрипты:
* **Работа со списками:** Сортировки, фильтрация и манипуляции с массивами данных.
* **Циклы и условия:** Логические задачи (FizzBuzz, поиск простых чисел, работа с `while` и `for`).
* **Функции:** Примеры декомпозиции кода для повторного использования.
* **Работа с файлами:** Скрипты для чтения и записи текстовых данных.

## 🛠 Технологии
* **Python 3.x** (Чистый стандартный функционал без тяжелых библиотек)
* **Linux Mint Terminal** (Запуск и тестирование всех скриптов)
